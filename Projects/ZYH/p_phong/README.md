# Phong光照模型演示项目

## 项目简介
本项目演示了如何使用Phong光照模型在OpenGL中渲染3D模型。

## 项目结构
- `main.cpp`: 主程序文件，包含窗口初始化、摄像机控制、模型加载和渲染逻辑
- `../../../shaders/phong.vs`: Phong光照模型的顶点着色器
- `../../../shaders/phong.fs`: Phong光照模型的片段着色器

## 实现功能
1. Phong光照模型（环境光、漫反射、镜面反射）
2. 3D模型加载与渲染
3. 摄像机控制（键盘和鼠标）
4. ImGui UI显示（FPS和摄像机位置）

## 编译与运行
1. 使用Visual Studio打开项目
2. 确保已配置好OpenGL环境和依赖库
3. 编译并运行项目

## 使用说明
- **WASD**: 前后左右移动摄像机
- **鼠标**: 控制摄像机视角
- **ESC**: 退出程序

## 技术细节
- 使用Assimp库加载GLB格式的3D模型
- 使用Phong光照模型实现真实的光照效果
- 支持纹理映射和材质属性设置
- 实现了深度测试和混合功能

## 模型来源
项目使用了swimming_pool_3d_scene.glb模型作为演示
